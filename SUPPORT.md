# Support

## Getting Help

- **Bug reports**: Open an issue using the Bug Report template.
- **Feature requests**: Open an issue using the Feature Request template.
- **Documentation**: Read the [README](./README.md) and [ARCHITECTURE.md](./ARCHITECTURE.md).
- **Discussions**: Use [GitHub Discussions](https://github.com/sparshsam/openjournal/discussions) for questions and community support.

## Security

If you discover a security vulnerability, please do NOT open a public issue.
Instead, review our [SECURITY.md](./SECURITY.md) for responsible disclosure instructions.
